# SaaS Landing Demo

## What is this
Minimal SaaS landing demo built with Next.js App Router.

## Why it exists
To demonstrate clean architecture, i18n, SEO and MVP discipline.

## Demo
/en  
/en/pricing  
/en/faq  
/en/contact  

## Tech stack
- Next.js (App Router)
- Tailwind CSS
- Route-based i18n

## How to demo (3 minutes)
1. Open Home
2. Check Pricing
3. Switch locale
4. Submit contact (mock)

## Limitations
- No backend
- No database
- No animations
- No analytics

## Future ideas
- Real contact submit
- CMS
- Analytics


## Who is this for
Founders and teams who need a fast, SEO-ready SaaS landing.

## What you see in 3 minutes
- Clean UI
- Pricing logic
- i18n routing
- SEO setup

## What is intentionally missing
- Backend
- Payments
- Analytics
